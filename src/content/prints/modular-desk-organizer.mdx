# Modular Desk Organizer

## Print Specifications

- Print Time: 6 hours
- Material: PLA
- Infill: 20%
- Layer Height: 0.2mm

## Features

- Stackable components
- Pen holder module
- Cable management module
- Small parts drawer
- Phone/tablet stand

## Assembly Instructions

Modules snap together using dovetail joints. No additional hardware required.
