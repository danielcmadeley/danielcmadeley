# Task Management CLI

## Overview

A command-line interface tool for managing daily tasks and projects, built with Rust for maximum performance.

## Features

- Task creation and management
- Priority levels
- Due dates
- Categories
- Export to various formats

## Technical Details

Built using Rust with focus on performance and memory safety.
